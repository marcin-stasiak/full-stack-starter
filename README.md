# Starter

## Description

NestJS, TypeORM, NextJS, TailwindCSS 

## Installation

```bash
$ npm install
```

## Configuration
```bash
# copy and edit .env file
$ cp .env.dist .env
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
